# Landing Page

Este proyecto consiste en el desarrollo de una Landing Page para un portafolio personal, desarrollada para el IFTS 29 para la materia de Desarrollo Web Front End. Se utilizaron tecnologías de HTML5 y CSS3 siguiendo una estructura semántica y diseño responsivo.

## Estructura del proyecto

- **Archivo "index.html"** ubicado en la raíz del proyecto.
- **Carpeta "css"**: Visual limpio y profesional
- **Carpeta "img"**: Se adapta a dispositivos móviles
- **Archivo "README.md"**: Rápida carga y SEO-friendly

- Repositorio en GitHub creado y proyecto subido.

## Uso de Google Fonts

- Enlace a Google Fonts incluido en el head del HTML.
- Tipografía aplicada correctamente en el sitio.
- Decisión de diseño: Elegí una fuente tipo Sans-Serif (Roboto/Montserrat) porque transmite una imagen profesional, limpia y es de fácil lectura en dispositivos digitales.

## Requerimientos HTML

- Declaración DOCTYPE y atributo lang="es".
- Metaetiquetas obligatorias: charset y viewport.
- Título descriptivo definido en la pestaña del navegador.
- Secciones de Main con IDs obligatorios: #sobre-mi, #tarjetas, #habilidades, #contacto y #peliculas.
- Barra de navegación (nav) con al menos 3 enlaces funcionales.
- Comentarios: Se incluyeron al menos 4 comentarios explicando bloques de código.

## Consideraciones de Accesibilidad y Privacidad

- Accesibilidad: Uso de atributos alt descriptivos en todas las imágenes.
- Privacidad: Siguiendo las instrucciones, se optó por un avatar/imagen alternativa para resguardar la identidad personal en el enlace público.
